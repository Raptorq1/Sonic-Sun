# Sonic-Sun

Welcome to sonic sun a mania inspired fan game made by me in the godot game engine use sonic worlds next as a base

you can check out the engine used to make this game here
https://github.com/Techokami/SonicWorldsNext

install directions for now are

You will need godot game engine to play

1. install godot 4 game engine

2. Extract the zip to a directory

3. Open godot game engine  

4. Click on the button that says scan and select the folder  after the project gets added  to the projects list click on the button that says edit this will import the files once the files have been imported Close godot 4 and reopen it select the project and click run to play
